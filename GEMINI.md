# Budget Manager

## Project Overview
A React-based budget management application.

## Tech Stack
- **Frontend:** React (Vite)
- **Styling:** Vanilla CSS (App.css, index.css)
- **Icons:** SVG sprite (public/icons.svg)

## Conventions
- Use functional components and hooks.
- Use Redux Toolkit.
- Avoid prop drilling.
- Adhere to the existing project structure in `src/`.
- Maintain the established naming conventions and styling patterns.
